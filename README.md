# STAT107-NBA-Project
database_24_25.csv - refers to the Kaggle NBA Team data set, provides all statistics and data we need for the Midway Report and analysis, includes team data, shooting percentage, three point percentage, three point frequency, wins, and losses

STAT107MidwayReport.Rmd - refers to midway report itself and its Rmarkdown file, no longer used for the Final Report, refer to the FinalReport.Rmd for the actual finished report

FinalReport.Rmd - final project report, describes and analyzes the NBA Team data set

README.md - the current file being viewed, describes what each of the files are and their purpose are

00_requirement.R - compiles and lists all of the R source libraries used, used in STAT107MidwayReport and FinalReport.Rmd so we don't have to repeatedly load libraries
